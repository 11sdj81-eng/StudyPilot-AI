# StudyPilot AI Sample Outputs

This folder contains lightweight demo artifacts for GitHub and recruiter review.

Included:

- `StudyPilot_v5_Review_Demo.pdf` - compact demo PDF output.
- `StudyPilot_v5_Review_RealDemo.md` - markdown demo generated from the real pipeline.
- `demo_manifest.json` - demo artifact manifest.
- `final_demo_report.md` - final demo run summary.
- `figure_status_report.json` - figure availability summary.

Large source textbooks, uploaded course materials, vector stores, model weights, and generated datasets are intentionally excluded.
